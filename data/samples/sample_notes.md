# Hackathon Idea — Memory Engine

## Core Concept
Build an AI that reconstructs forgotten context from your digital life.
Users upload screenshots, chats, and notes. The AI finds connections.

## Tech Stack
- RAG architecture for retrieval
- ChromaDB for vector storage
- OpenAI or local embeddings
- FastAPI backend, Streamlit frontend

## Potential Use Cases
- Developers reconstructing project context after a break
- Students reviewing study sessions
- Researchers connecting ideas across notes
